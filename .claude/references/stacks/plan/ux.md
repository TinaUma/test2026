# UX/UI Expert — Planning Reference

## Principles
1. **Семантика первична** — правильные HTML-теги важнее стилей
2. **Контраст 4.5:1** — минимум для текста, 3:1 для UI-элементов
3. **Focus visible** — каждый интерактивный элемент имеет видимый focus
4. **Токены вместо значений** — var(--space-4) вместо 16px
5. **Bootstrap utilities** — использовать классы, не писать CSS без нужды
6. **Reduced motion** — уважать системные настройки пользователя
7. **Progressive disclosure** — не перегружать интерфейс

## Code Patterns
No specific code patterns. Follow stack principles above.
