# Python System Developer — Planning Reference

## Principles
1. **Type hints везде** — mypy совместимость
2. **Async для I/O** — не блокировать event loop
3. **Multiprocessing** — для CPU-bound задач
4. **Context managers** — для ресурсов
5. **Graceful shutdown** — signal handling
6. **Typer для CLI** — type-safe аргументы
7. **Structured logging** — не print()

## Code Patterns
No specific code patterns. Follow stack principles above.
