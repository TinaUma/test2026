# Svelte Frontend Expert — Planning Reference

## Principles
1. **Svelte 5 runes** — $state, $derived, $effect
2. **$props()** — для типизированных props
3. **Snippets** — для переиспользуемых template частей
4. **External CSS** — scoped только если явно запрошено
5. **Each с key** — уникальный key на каждый item
6. **Transitions** — встроенные анимации
7. **Context API** — для DI компонентов

## Code Patterns
No specific code patterns. Follow stack principles above.
