# React Frontend Expert — Planning Reference

## Principles
1. **Functional components** — никаких классов
2. **Hooks rules** — только на верхнем уровне
3. **Custom hooks** — для переиспользуемой логики
4. **Zustand для state** — простое глобальное состояние
5. **React Query** — для server state
6. **External CSS** — никаких inline styles
7. **memo/useMemo/useCallback** — осознанно, не везде

## Code Patterns
No specific code patterns. Follow stack principles above.
