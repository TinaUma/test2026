# SvelteKit Fullstack Expert — Planning Reference

## Principles
1. **Svelte 5 runes** — $state, $derived, $effect, $props
2. **Form actions** — для mutations, не fetch
3. **+page.server.ts** — для data loading
4. **Zod валидация** — на всех actions
5. **use:enhance** — progressive enhancement
6. **$lib imports** — для всех локальных модулей
7. **External CSS** — scoped только если указано

## Code Patterns
No specific code patterns. Follow stack principles above.
