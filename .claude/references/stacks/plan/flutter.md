# Flutter Expert — Planning Reference

## Principles
1. **GetX для state** — controllers, reactive variables
2. **Bindings для DI** — dependency injection
3. **Freezed для models** — immutable data classes
4. **Const constructors** — где возможно
5. **Theme.of(context)** — для стилей
6. **ListView.builder** — для длинных списков
7. **Repository pattern** — для данных
