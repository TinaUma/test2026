# PHP Expert — Planning Reference

## Principles
1. **declare(strict_types=1)** — в каждом файле
2. **Prepared statements** — никакой конкатенации SQL
3. **Readonly classes** — immutable где возможно
4. **Enums** — вместо констант
5. **Match expressions** — вместо switch
6. **Constructor promotion** — меньше boilerplate
7. **PSR-4 autoloading** — Composer

## Code Patterns
No specific code patterns. Follow stack principles above.
