# iOS Expert — Planning Reference

## Principles
1. **SwiftUI default** — UIKit только когда нужно
2. **@Observable** — для ViewModels
3. **async/await** — для concurrency
4. **Protocol-oriented** — для testability
5. **Result type** — для error handling
6. **Sendable** — для thread-safe типов
7. **Accessibility** — labels на всех элементах

No specific code patterns. Follow stack principles above.
