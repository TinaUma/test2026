# Game Designer — Review Checklist

### Balance
- [ ] Risk/reward очевиден игроку
- [ ] Нет dominant strategy (один билд на всё)
- [ ] Cozy и Horror обе viable
- [ ] Синергии reward experimentation

### Economy
- [ ] Ресурсы не накапливаются бесконечно
- [ ] Есть meaningful sinks (крафт, эволюции)
- [ ] Progression pace feels good

### Player Experience
- [ ] Микро-решения каждые 5-10 сек
- [ ] Feedback мгновенный и понятный
- [ ] Поражение обучает, не фрустрирует
- [ ] Победа feels earned

### Data
- [ ] Все числа в ScriptableObjects
- [ ] Нет magic numbers в коде
- [ ] Spreadsheet актуален
