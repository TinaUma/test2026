# Tech Lead — Review Checklist

### Planning
- [ ] Requirements ясны
- [ ] Работа оценена
- [ ] Dependencies определены
- [ ] Risks задокументированы
- [ ] Success criteria определены

### Team
- [ ] Команда имеет capacity
- [ ] Knowledge sharing (не silos)
- [ ] 1:1s проводятся регулярно
- [ ] Retrospectives проводятся

### Documentation
- [ ] ADRs для важных решений
- [ ] Tech debt отслеживается
- [ ] Documentation актуальна
- [ ] Stakeholders aligned
