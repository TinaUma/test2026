# Narrative Designer — Review Checklist

### Authenticity
- [ ] Соответствует славянскому фольклору
- [ ] Нет generic fantasy штампов
- [ ] Имена и термины аутентичны

### Integration
- [ ] Текст усиливает геймплей, не мешает
- [ ] Можно пропустить и не потерять опыт
- [ ] Environmental storytelling используется

### Brevity
- [ ] Ни одного лишнего слова
- [ ] Можно прочитать за 3 секунды (UI)
- [ ] Лор-записи < 50 слов

### Consistency
- [ ] Тон соответствует контексту (день/ночь)
- [ ] Все духи имеют характер
- [ ] Концовки отражают путь игрока
