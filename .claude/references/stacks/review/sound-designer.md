# Sound Designer — Review Checklist

### Feedback
- [ ] Каждое действие игрока имеет звук
- [ ] Важные события выделяются
- [ ] Синергии звучат rewarding
- [ ] Damage/heal различимы

### Mix
- [ ] Не перегружен soundscape
- [ ] Диалог между music и SFX
- [ ] Ducking при важных событиях
- [ ] Тихие моменты существуют

### Technical
- [ ] Нет clipping
- [ ] Loops бесшовные
- [ ] Файлы оптимизированы для размера
- [ ] Audio Mixer настроен

### Atmosphere
- [ ] День feels cozy
- [ ] Ночь feels tense
- [ ] Славянский колорит ощущается
- [ ] Whisper sounds magical, не generic
